# WebGoat

WebGoat is an intentionally vulnerable web application developed by OWASP for learning and practicing web application security concepts.  

It provides hands-on labs covering real-world vulnerabilities such as:
- Broken Access Control
- SQL Injection
- Authentication Flaws
- Session Management
- SSRF
- XXE
- Insecure Deserialization

This repository documents practical WebGoat exercises, exploitation techniques, lessons learned, and mitigation strategies based on OWASP Top 10 concepts.
